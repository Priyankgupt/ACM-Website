Alibaba Cloud AI Challenge 2021


Alibaba Cloud Academy had partnered with IndiaSpark for one of Prestigious Program called as -Global AI Forward Challenge,2021.This program is applicable for developers, students of any stream and semester- Both Technical & Non-technical STEM students, researchers, Faculty, enterprises etc. This program is totally free of cost. About the Challenge: Through learning about cloud computing and obtaining industry-standard certifications, you will be able to progress in your career. The Alibaba Cloud Academy offers a range of certifications fitting your desired education path and learning level. Get on the path to your next great journey in the cloud with Alibaba Cloud Certifications. This Challenge will:

• You will have a resource site for all students exploring tech careers, even those with non-technical background.
• Ace your career with the power of AI and the global exposure extended by Alibaba Cloud community.

WEB-D Competition


Web-D Competition was an eight-hour-long event where students were introduced to the basics of Web Development, so they can practice it on their own with their creativity and innovative mind. The Speaker for the event was Mr. Ayush Gupta, vice-chair, ACM. He introduced the students to the concepts of Web Development and explained the related terms in detail. The session started off with an introduction to HTML, a brief about how we can create a simple web page using this language. Mr. Gupta briefed the students about a variety of features and tools that can be used for Web Development. He highlighted the key points to be considered while making the layout. Next, he explained everything, beginning from using the appropriate IDE and ending with how to successfully build a responsive website. Also, the students were thoroughly explained about CSS (Cascading Style Sheet) from the very basics. He not only demonstrated the design but also introduced and made students aware of the various methods as to how we can improve the look of our website.
